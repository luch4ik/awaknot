
# Simple Alarm App


An Alarm App copy cat implemented using [AlarmKit](https://developer.apple.com/documentation/alarmkit) introduced in WWDC 2025.


For more details:

- [SwiftUI: Alarm App CopyCat with AlarmKit (WWDC 2025)-Part 1](https://medium.com/@itsuki.enjoy/swiftui-alarm-app-copycat-with-alarmkit-wwdc-2025-part-1-27fad3186791)
- [SwiftUI: Alarm App CopyCat with AlarmKit (WWDC 2025)-Part 2](https://medium.com/@itsuki.enjoy/swiftui-alarm-app-copycat-with-alarmkit-wwdc-2025-part-2-5c3cb2194c54)


## Capability
- Schedule a traditional alarm, one shot or repeat on a schedule
- Schedule a traditional timer, able to repeat, displaying the countdown on the Lock Screen, Dynamic Island, and within our app
- A fully customized alarm, either on a fixed date or on a relative schedule, maybe with a countdown timer, maybe with a snooze
